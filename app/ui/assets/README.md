## Asset Guidelines

### SVG Icons
- Size: 24Ã—24px base size
- Stroke: 2px
- Color: Single color (will be styled via CSS)
- Optimize: Run through SVGO

### Image Formats
- Icons: SVG (vector)
- Raster images: PNG with transparency
- Avoid: JPG (unless photo assets)
